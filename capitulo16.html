Capítulo 15: Proyecto final

🎯 Objetivo

Construir una librería de componentes Web funcional, reutilizable y bien documentada con Stencil.js. Se aplicará todo lo aprendido durante el curso.

🧱 Requisitos mínimos

Tu librería debe incluir:

  • Al menos 4 componentes reutilizables
  • Estilos encapsulados (shadow DOM)
  • Props bien definidas y documentadas
  • Al menos un evento personalizado
  • Slots y uso del ciclo de vida
  • Test unitarios en cada componente
  • Historias en Storybook con controles
  • Build optimizada y publicada localmente

💡 Propuesta de componentes

  • <mi-boton>: botón configurable con slots e iconos
  • <mi-alerta>: alertas con diferentes variantes (info, error, etc.)
  • <mi-tarjeta>: componente para mostrar contenido con estilo
  • <mi-modal>: ventana modal controlada por props y eventos

Puedes adaptarlos o crear los tuyos propios.

📁 Estructura del proyecto


mi-ui/
├── src/
│   └── components/
│       ├── mi-boton/
│       ├── mi-alerta/
│       ├── mi-tarjeta/
│       └── mi-modal/
├── dist/
├── .storybook/
├── www/
├── package.json
├── stencil.config.ts

📖 Documentación con Storybook

  • Incluye al menos una historia por estado del componente
  • Usa args y argTypes
  • Documenta qué hace cada prop y evento

🧪 Pruebas unitarias

  • Testea props y comportamiento interno
  • Simula eventos y verifica su emisión
  • Usa @testing-library/stencil o el setup estándar de Jest

🚀 Publicación local

  1. Ejecuta npm run build
  2. Haz npm pack para generar el paquete
  3. Instálalo en otro proyecto con npm install ./mi-ui-1.0.0.tgz

🌐 Despliegue opcional

Puedes desplegar tu documentación con Storybook en GitHub Pages, Vercel o Netlify para que sea accesible públicamente.

🎓 Ejercicio final

  1. Crea tu librería desde cero o parte del proyecto del curso
  2. Diseña al menos 4 componentes reutilizables
  3. Aplica decoradores, ciclo de vida, slots y eventos
  4. Testea y documenta con Storybook
  5. Empaqueta y prueba su publicación local
  6. Entrega el repositorio o proyecto como prueba final del curso
